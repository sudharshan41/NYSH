## 🚀 Getting Started

Follow these steps to run the project locally:

###  Clone the repository

```bash
https://github.com/sudharshan41/NYSH.git
```

### Navigate to the project directory
```bash
cd NYSH
```

### Install dependencies
```bash
npm install
```
### Start the development server
```bash
npm run dev
```
### 🌍 Open Your Browser
Visit:

``` bash
http://localhost:9002/
```
